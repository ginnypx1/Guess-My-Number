Guess My Number

Play a simple game of Guess My Number.

I created this version of Guess My Number to help me practice Python in several forms.

You can see the evolution of the project in my Github repository:

  • guess_my_number.py plays a simple game in the command line.
  • tkguess.py plays Guess My Number in a Tkinter widget.
  • djguess runs a django version of Guess My Number.
